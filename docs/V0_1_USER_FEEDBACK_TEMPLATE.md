## 8. 判断边界

请注意：

- 本地同屏游戏不能真正变成 LAN，通常应走 Steam Remote Play 或 Sunshine + Moonlight。
- Steam P2P / 官方服务器限定游戏不一定适合 n2n。
- 单机测试通过不等于真实双机一定通过。
- 未审核 adapter 可能需要修正端口、启动参数或适用条件。
